# Catatan Belajar CLI dengan Go

Halo! Ini adalah kumpulan folder latihan saya selama mempelajari bahasa pemrograman Go, khususnya dalam pembuatan aplikasi berbasis terminal (CLI).

## Tentang Proyek Ini
Beberapa proyek di sini mungkin terlihat dibuat dalam waktu singkat, namun sebenarnya logika dan konsep CLI ini sudah saya pelajari cukup lama secara mandiri. 

Dulu saya lebih fokus belajar koding secara offline karena belum paham cara menggunakan GitHub dan alur belajar saya saat itu masih belum terarah. Sekarang, saya coba mendokumentasikan hasil belajar ini supaya lebih rapi dan bisa saya buka-buka lagi di kemudian hari.

## Progres Latihan
Catatan ini saya buat untuk bahan belajar kembali jika suatu saat saya lupa:
* **Dasar:** Latihan sintaks dan logika sederhana.
* **Upgrade:** Belajar merapikan kode dengan memisahkan fungsi ke package lain.
* **Kompleks:** Belajar menggunakan pointer dan struktur folder (model, helper, fitur) supaya lebih teratur.

## Tujuan Dokumentasi
Tujuan utama saya hanya untuk dokumentasi pribadi. Karena saya sadar terkadang saya masih sering lupa dengan apa yang sudah dipelajari, catatan ini sangat membantu saya untuk mengingat kembali konsep-konsep yang ada, termasuk soal penggunaan pointer dan struktur kode.

## Catatan Pribadi
Saya mohon maaf jika dokumentasi atau kodingan ini masih banyak kekurangan. Saya masih terus belajar, dan kalau saya lupa, tidak apa-apa bagi saya untuk membuka kembali catatan ini atau diingatkan lagi. Segala kekurangan di sini adalah bagian dari proses belajar saya agar bisa lebih baik lagi ke depannya.

Semoga catatan sederhana ini bisa bermanfaat buat saya sendiri di masa depan.
